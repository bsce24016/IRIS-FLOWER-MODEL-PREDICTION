# IRIS-FLOWER-MODEL-PREDICTION
# Iris Flower Classification

## Overview

The **Iris Flower Classification** project is a Machine Learning classification system that predicts the species of an iris flower based on its physical measurements. The model uses features such as **sepal length, sepal width, petal length, and petal width** to classify flowers into one of three species:

* **Iris-setosa**
* **Iris-versicolor**
* **Iris-virginica**

This project is one of the most classic beginner-friendly machine learning problems and serves as a great introduction to **classification algorithms, data preprocessing, model training, and performance evaluation**.

## Features

* Classifies iris flowers into three species
* Uses flower measurements as input features
* Applies **supervised machine learning** for prediction
* Includes data preprocessing and label encoding
* Evaluates model performance using **accuracy, classification report, and confusion matrix**
* Helps understand the complete workflow of an ML classification project

## Machine Learning Workflow

1. **Dataset Loading**
   Load the Iris dataset containing flower measurements and species labels.

2. **Data Exploration**
   Inspect the dataset using:

   * first 5 rows
   * dataset information
   * missing values check

3. **Feature and Target Separation**

   * Input features: sepal length, sepal width, petal length, petal width
   * Target variable: flower species

4. **Label Encoding**
   Convert categorical species labels into numerical form for model training.

5. **Train-Test Split**
   Split the dataset into training and testing sets.

6. **Model Training**
   Train a classification model such as **K-Nearest Neighbors (KNN)** to learn the relationship between flower measurements and species.

7. **Evaluation**
   Evaluate the model using:

   * Accuracy score
   * Classification report
   * Confusion matrix

## Project Objective

The objective of this project is to build a model that can accurately classify iris flowers based on their measurements. It also serves as a practical introduction to **supervised learning**, **classification tasks**, and **model evaluation techniques**.
